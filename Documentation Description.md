# Anonymous-Students-Support-Services
I've created a comprehensive multi-service student support platform for my internship project.

Quick Start
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
Documentation
Document Description
SYSTEM_DOCUMENTATION.md
Technical documentation, architecture, database schema
USER_MANUAL.md
Student and administrator usage guide
Technology Stack
•	Frontend: React 18 + TypeScript + Tailwind CSS
•	Backend: Supabase (PostgreSQL)
•	Hosting: Netlify
Environment Variables
Create a .env file:
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-anon-key
Features
•	Anonymous student reporting with AI emotion detection
•	3D Gamified career development (Career Clues)
•	Mental health toolkit AI driven (eMindCare)
•	Points and voucher reward system
•	Multi-department admin dashboard
•	Chatbot Managing by the Super Admin for FAQs
